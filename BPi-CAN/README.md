BPI-Model-Railroad-Interface
============================

This is the hardware part of the BPi Railroad Interface:

[Deutsch Beschreibung][http://lnxpps.de/bpi/]
[englisch description][http://lnxpps.de/bpi/]


Interfaces 
----------

- CAN
- I2C
- RS232 Console
- RS232
- RS485
- Sniffer input
- S88
- SPI for display
