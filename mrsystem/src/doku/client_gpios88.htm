Daemonmrgpios88

Parameter

Kommandozeilenparameter sind:

ParameterBeschreibung
-b <bcast_addr/int>broadcast address or interface - default 255.255.255.255/br-lan
-i [0|1]invert signals - default 0 -> not inverting
-d <event id>using event id - default 0
-e <hash>using CAN
-m <s88modules>number of connected S88 modules - default 1
-o <offset>addressing offset - default 0
-s <port>second destination port of the server
-t <time in usec>microtiming in usec - default 15 usec
-frun in foreground (for debugging)
-vbe verbose

Beschreibung

Der Daemon mrgpios88 fragt auf dem BPi über GPIOs Rückmelder auf dem den S88 Bus von Märklin ab.
