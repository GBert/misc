Client_CS2ETH

Daemon

Daemonmrcs2eth
Projektclient_cs2eth

Parameter

Kommandozeilenparameter sind:

ParameterBeschreibung
-a <addr>Adresse der Servers, mit dem eine Verbindung aufgebaut werden soll.
-hFiltern von CAN nacrichten, damit die MS2 keinen Master sieht.
-i <iface>Interface, auf dem auf Verbindungen von anderen Clients zu drehscheibe gewartet wird.
-bUDP Nachrichen werden als Broadcast versendet.
-p <port>Port der Servers, mit dem eine Verbindung aufgebaut weden soll
-foptional, im Vordergrund starten (don't fork)
-voptional, mit zusätzluchen Ausgaben (verbose) starten
-tZusätzliche Traceausgaben für die Can Nachrichen auf der UDP Seite. Nur verfügbar, wenn bein compilieren TRACE definiert ist.
-?optional, gibt eine kurze Hilfe aus

Beschreibung

Der Daemon mrcs2eth2 wertet Nachrichten vom Ethernet Interface an den UDP Port der CS2 aus verschickt Nachrichten aus dem System als Broadcast im Format der CS2.
