Parameter

Kommandozeilenparameter sind:

ParameterBeschreibung
-a <addr>Adresse, auf der auf Verbindungen von clients gewaret wird
-i <iface>Interface, auf dem auf Verbindungen von anderen Clients zu drehscheibe gewartet wird.
-p <port>Port, auf der auf Verbindungen von clients gewaret wird
-foptional, im Vordergrund starten (don't fork)
-voptional, mit zusätzluchen Ausgaben (verbose) starten
-?optional, gibt eine kurze Hilfe aus

Beschreibung

Der Daemon drehscheibe hört auf der Adresse, die mit -a, und dem Port, der mit -p in der Kommandozeile übergeben wird, auf eingehende Verbindungen der Clients.

Ist eine Verbindung mit einem Client etabliert, werden sämtliche Messages dieses Clients an sätliche anderen Clients weiter geleitet.
