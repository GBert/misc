Cheap PCIe Interface for Microchip PICs
=======================================

|  R3  |  R4  | PCI ID  | Function            |
| ----------- | --------|---------------------|
| open | open | 0x3253  | 2 serial            |
| short| open | 0x3250  | 2 serial 1 parallel |
| short| short| 0x3050  | 1 parallel          |
