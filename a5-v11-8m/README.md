====================

scripts/feeds install luci-app-openvpn

Create image
------------
```
# mtd0.img u-boot
# mtd1.img u-boot-Env
# mtd2.img factory
cat u-boot.img u-boot-env.img factory.img lede-ramips-rt305x-a5-v11-8m-squashfs-sysupgrade.bin > combined_image.img
```
