CAN-Geräte

Hier können alle über den CAN-Bus angebundenen und erkannten Geräte eingesehen und konfiguriert werden. Unter anderem können z.B. die Verbrauchswerte der Gleisbox betractet werden.

Über das Dropdown-Menü können die erkannten Geräte ausgewählt werden. Anschließend werden alle vom Gerät übertragenen Informationen dargestellt.

Lokbilder

Hochladen:

Über diese Option lassen sich Bilder von dem benutzen Gerät auf dem Server hochgeladen werden.

Download:

Alternativ können Bilder unter angabe des Links direkt aus dem Internet herunter geladen werden.

Hinweis: Aktuell können die Bilder noch nicht über die Benutzeroberfläche einer Lok zugewiesen werden. Dieses Feature ist noch in der entwickung.

Server-Einstellungen:

In dieser Sektion können einige allgemeine Einstellungen des Servers verändert werden.

Server Neustarten:

Der Server wird heruntergefahren und startet anschließend neu.

Server Herunterfahren

Der Server wird heruntergefahren und muss für eine erneute Verwendung manuell neu gestartet werden.

Gleisprotokoll:

Es können wahlweise die Protokolle Märklin Motorola, DCC und MFX bzw. M4 einzeln aktiviert oder deaktiviert werden.

Neuanmeldezähler setzen:

Der MFX-Neuanmeldezähler wird erhöt, um eine ggf. verloren gegangene Lok wieder zu finden. Sollte die Gleisspannung eingeschaltet sein, so wird diese für eine halbe Sekunde ausgeschaltet.
